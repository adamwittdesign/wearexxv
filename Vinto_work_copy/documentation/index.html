HTML Structure

This theme is build with help of Twitter Bootstrap. Lots of Bootstraps features are supported in this theme, so a lot of the documentation for this theme can be found at http://twitter.github.io/bootstrap/index.html.
The following Bootstrap features are not supported:

  • Icons by Glyphicons
  • Button groups and dropdowns
  • Thumbnails
  • Hero unit
  • Media component
  • Dropdowns
  • Carousel
  • Responsive for large desktops (> 1200px)
  • JavaScript: Dropdowns
  • JavaScript: Scrollspy
  • JavaScript: Buttons
  • JavaScript: Carousel
  • JavaScript: Typeahead

Section Types

There are 6 different section type classes:

  • Header Type - .header-type
  • Page Type - .page-type
  • Separator Type - .separator-type
  • Title Type - .title-type
  • Carousel Type - .carousel-type
  • Footer Type - .footer-type

These are different classes that are meant to be added to a section (page wrap). Besides these type classes you can easily change the background by adding the following classes:

Different Backgrounds

  • Dark Background - .dark-bg
  • Grey Background - .grey-bg
  • Lighter Grey Background - .grey2-bg

Different Main Font Color

  • White Font - .font-white

Icons

Vinto comes with the integration of the Fontawesome icon set. You can go to the site of Fontawesome to see how to easily add icons to your HTML markup. Vinto includes some styling specially for the Fontawesome icons, for example in combination with the .shape and special buttons .btn-icon.

Buttons

Vinto comes with a special type of button: .btn-border. The markup for this button is as follows: <a href="#" class="btn btn-border">Normal Button</a>. The default color of this button is set to #636363, but you can add the following classnames to simply change the color: border-white.

Another type of button that Vinto has styled is the .btn-primary, which has the following color code #f00423.

Vinto also includes diamond buttons with icons .btn-icon which standard has a light grey background. You can add the following class .btn-icon-white to have a white background.

CSS3 Animations

Adding a CSS3 animation to an element is simple with Vinto. You only have to add the following classnames: animated delayed no-opacity, and the following two attributes to your div: data-animation and data-animation-delay. Animate.css is implemented in Vinto, so all the animations from https://daneden.me/animate/ are supported. Also the following animation: ts-slideInUp has been added to Vinto. Each individual animation on an element can be given a custom animation-delay with the data-animation-delay attribute.

AJAX Popup

The two folders where AJAX content is loaded from are hard coded: portfolio-item and team-member. To add an ajax popup to an anchor tag you'll have to do the following things:

  1. Create your page that will be dynamically loaded inside the popup. There are two types of content: a portfolio item (foldername: portfolio-item) and a team member (foldername: team-member). Best is to duplicate a demo file inside these folders and adjust the content to your likings, this way you'll have the exact markup that's needed for your page. From your page everything inside the div with the unique idname: project-inner will be loaded into the AJAX popup.
  2. Add the classname ajax-nav to your anchor tag.
  3. Add an href to the anchor tag which is the url to the content you want to dynamically load with AJAX.

There are two preset types of styling for the ajax popup: style-one and style-two.

Note that in order for the AJAX popups to work, you'll have to view your website on an online server.

Twitter Feed

Vinto comes with an option to display your Twitter Feed. Just create an empty div with classname tweet and the following attributes: data-twitter-username (twitter username to display feed) and data-count (the amount of tweets to display).

Inside assets/php/twitter/index.php you'll have to add your Twitter App Consumer Key, Twitter App Consumer Secret, Twitter App Access Token and Twitter App Access Token Secret. To get these codes you need to have a twitter App for your usage in order to obtain OAuth credentials, see https://dev.twitter.com/apps for help.

Checkout https://github.com/StanScates/Tweet.js-Mod for more information about Tweet.js-mod.

AJAX Contact Form

Vinto comes with a working AJAX contact form. The contact form exists of two files: contact-form.js (assets/js/) and contact.php (assets/php/contact-form/). Inside contact.php you can specify to which email address the message has to be sent. Also the markup of the message can be modified inside the php file.

Note that in order for the AJAX contact form to work, you'll have to view your website on an online server.
